## general
#### NORMAL
`jk` - `<Esc>`
`;` - `:`

## completions
`<C-b>` - scroll docs down
`<C-f>` - scroll docs up
`<C-Space>` complete
`<C-e>` - abort
`<CR>` - confirm selected
`<Tab>` - select next item
`<S-Tab>` - select prev item

## lsp
#### NORMAL
`K` - show hover docs
`<leader>gd` - go to definition
`<leader>gr` - go to references
`<leader>ca` - code action
`<leader>rn` - rename all

## null-ls
`<leader>gf` - auto format file

#### NORMAL

`<leader>ma` - annotate
